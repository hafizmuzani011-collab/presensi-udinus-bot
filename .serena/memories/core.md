# Core System Map

This project is an automated academic assistant bot for Udinus University students (supporting student "saya" and spouse "pacar"). It scrapes Moodle (Kulino) for tasks and materials, and SiAdin (MHS portal) for attendance, grades (KHS), schedules, and exams.
